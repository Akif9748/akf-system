# Nodejs system info
You can write your system info to your console like Neofetch.
But only for Windows for now.
It can show:

- ASCII art for your OS
- Your username
- OS name
- Build name
- Host name
- Uptime of system
- CPU name
- RAM usage
- GPU name
- Screen Resolution
