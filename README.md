# akf-system
You can write your system info to your console like Neofetch.
It can show:

- ASCII art for your OS
- Your username
- OS name
- Build name
- Host name
- Uptime of system
- CPU name
- RAM usage
- GPU name 
- Screen Resolution

You can install it:
```
npm i . -g
```
Then:
```
akf-system
```
