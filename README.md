# akf-system
A Node.js based system-information writer.
It can show with order:

- ASCII art for your OS
- Your username
- Host name
- OS name
- Build name
- Uptime of system
- CPU name
- RAM usage
- Motherboard
- GPU name 
- Screen Resolution

You can install it with `npm i -g akf-system` and use with `asys`

Or you can use it without install via `npx akf-system`
