# Nodejs system info
You can write your system info to your console like Neofetch.
It can show:

- ASCII art for your OS (Only for Windows)
- Your username
- OS name
- Build name
- Host name
- Uptime of system
- CPU name
- RAM usage
- GPU name (Only for Windows)
- Screen Resolution
